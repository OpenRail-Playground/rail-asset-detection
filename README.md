# Rail Asset Detection

Detecting rail assets such as the blue platform displays on the station platforms.

### Tasks 

#### Machine Learning
* Convert Images
* Label Images
* Train object detection model

#### Extract Coordinates
* Get the geo coordinates from the detected objects
